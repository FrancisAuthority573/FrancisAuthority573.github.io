C++

Introduction to c++

C++ is a cross-platform language that can be used to create applications. C++ was developed by Bjarne Stroustrup, as an extension to the C language.

C++ is close to C but the difference between the both is that, C does not support object and classes while C++ do support classes and object.

Installation and setting up the C++ IDE

An IDE is an acronym of Integrated Development Environment. It can be a software for coding and programming that interprets or compile any code with a compiler or interpreter.

C++ uses a popular compiler which is GCC. In installing and setting up your IDE you need to have a text editor like notepad++ where you can write all the codes and a compiler like GCC for compiling the code which gives the result or output.

There are many text editors and compilers but we will be using and IDE (Integrated Development Environment) which we will use to write and run the code.

The C++ IDEs which are popular are Visual Studio, Dev c++, Eclipse, code::block, etc. But we are going to be using Dev c++.

If you don't have Dev c++ you can download it from here, or use another C++ IDE like Visual Studio or code::block.
